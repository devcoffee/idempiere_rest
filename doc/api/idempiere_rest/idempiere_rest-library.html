IdempiereClient
The main class to do all requests to idempiere-rest API developed following the singleton pattern, once logged in, can be called in every part of the applications. Once instanciated, setBaseUrl must be the first method called. After that, login or oneStepLogin can be called, to retrieve the token. This class holds all Session information.
